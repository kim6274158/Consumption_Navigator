## 2. 핵심 서비스 개요

Olla는 개인화된 소비 분석, 맞춤형 재무관리 컨설팅, AI 소비 스토리 생성, 실시간 금융 혜택 안내를 통합 제공하는 스마트 금융 코치 서비스입니다

사용자의 실제 결제 내역을 기반으로 AI가 자동으로 소비 패턴을 분석 및 진단하고, 절약, 맞춤 혜택, 구독 등 금융 전반의 최적안을 실시간으로 제안합니다.또한, 감성 기반의 소비 일기와 스토리텔링 서비스로 지속적인 재무 개선 경험을 제공합니다.

## 3. 제안 배경 및 목적

### 💡 제안 배경
청년층, 디지털 취약계층, 고령자 등은 자신의 소비 흐름을 정확히 이해하기 어려워 재무적 위험에 노출되기 쉽습니다. 카드 혜택, 금융상품 관리의 복잡성이 개인의 역량에 과도하게 의존하면서 국민 전반의 금융문해력 불균형이 심화되고 있습니다. 이에 누구나 소비와 재무 상태를 쉽게 관리할 수 있도록 돕는 투명한 AI 기반 금융 관리 서비스의 필요성이 증가했습니다.

### 🎯 목적
AI와 데이터 기반 기술을 활용하여 국민 개개인이 소비를 투명하게 이해하도록 돕고, 불필요한 지출을 줄여 , 스스로 재무 건전성을 관리할 수 있는 자기주도적 금융관리 환경을 조성하는 것입니다.

## 4. 적용 기술 및 아키텍처

### 1) 활용 데이터 및 AI 모델
* **활용 데이터**: 신용카드·체크카드 및 은행 거래 내역 데이터, 계좌 잔액 데이터, 카드 목록 데이터, 카드 상품 정보 데이터 (카드고릴라 스크래핑) 

### 2) MCP(Multi-Channel Platform)를 활용한 금융 데이터 통합 및 확장성
* **구현 내용**: 신용카드/체크카드 거래내역, 계좌 잔액, 카드 상품 정보 등 다양한 금융 데이터소스를 MCP 서버로 통합 관리합니다. 증권, 보험 등 추가 금융상품 연동 시 새로운 서버만 추가하면 되는 **확장 가능한 아키텍처**를 구축합니다.
* **기대 효과**: 표준화된 프로토콜로 다양한 금융기관 API 연동이 용이하며, 서비스 확장 시 기존 시스템 수정 없이 새로운 데이터소스 추가가 가능합니다.

### 3) Reflexion Agent 기반 소비 패턴 분석 및 자기개선 시스템
* **구현 내용**: 초기 소비 분석 결과를 자체 피드백 루프를 통해 반복적으로 개선하여 분석의 정확도를 높입니다. 사용자 피드백을 반영하여 개인화된 분석 정확도를 지속 향상시킵니다.
* **기대 효과**: 단순 규칙 기반이 아닌 자기 학습하는 AI로, 사용자별 특성을 반영한 정교한 소비 분석 및 맞춤 추천이 가능합니다.

### 4) Dual Memory System 기반 맥락 인식 챗봇
* **Short-Term Memory**: 현재 대화 세션의 문맥 유지 ("아까 말한 그 카드" 이해) 
* **Long-Term Memory (벡터 DB)**: 사용자의 소비 성향, 선호도, 과거 상담 이력 저장 및 활용.
* **기대 효과**: 지속적 관계를 형성하는 '나를 아는 금융 코치' 경험을 제공하여 사용자 만족도 및 서비스 재사용률을 증가시킵니다.

### 5) 벡터DB 기반 카드 추천 엔진
* **구현 내용**: 사용자 소비 패턴 벡터와 카드 상품 혜택 벡터 간 유사도를 계산하여 사용자에게 **가장 가까운 거리의 최적 카드**를 추천합니다.

## 5. 주요 서비스 소개 (기능)

### 1. 소비 분석
* 사용자의 카드/거래 내역을 자동 분석하고 시각화 대시보드를 제공합니다.
* 카테고리별 목표 금액 설정 및 지출 관리가 가능합니다.
* AI가 소비 트렌드, 절약 가능 내역, 혜택/카드 추천 등을 자동 제안합니다.

### 2. 재무 관리
* 사용자 자산(은행, 카드, 증권, 보험 등) 연결 및 통합 관리를 지원합니다.
* 카테고리별 목표 금액 설정 및 지출 관리를 통해 예산 관리를 돕습니다.

### 3. 소비 일기 (개인화 & 감성 서비스)
* **AI 소비 일기 생성**: 결제 내역 기반으로 일기를 자동 생성하며, 소비 패턴으로 감정 분석(행복, 스트레스 등) 및 응원 메시지를 제공합니다.
* **소비 스토리텔링**: 주/월 단위 소비 여정을 '자기계발에 집중했어요'와 같은 스토리로 재구성하여 긍정적 동기를 부여합니다.

### 4. 카드 추천 및 혜택 알림
* **AI 카드 추천 엔진**: 실시간 소비 패턴 분석 후 최적 카드를 추천하고, 변경 시 연간 혜택 증가 시뮬레이션을 제공합니다.
* **놓친 혜택 알림**: 결제 시점에 놓친 할인 혜택 등을 실시간으로 알려주고, 할인 한도 잔여분을 안내합니다.

### 5. 챗봇/음성 비서
* Dual Memory System 기반 챗봇으로 실시간 질의응답 및 금융관리 피드백을 제공합니다.
